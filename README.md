# Nova Retail Sales Project

## 📌 Project Overview
This is an End-to-End Data Analytics Project created using Excel, SQL, and Power BI.

## 🛠️ Tools Used
- Microsoft Excel
- MySQL
- Power BI

## 📂 Files Included
- Nova_Retail_Sales.xlsx
- Nova_Retail_Sales_Dashboard.pbix
- Nova_Retail_sales_SQL_Project.sql
- PowerBI_Dashboard_Screenshot.png

## 📊 Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Total Customers KPI
- Profit Margin KPI
- Sales by Region
- Sales by Category
- Sales by Payment Mode
- Order Status Analysis
- Monthly Sales Trend

- Power BI Dashboard

- Data Visualization
